👩‍💻 Data Professionals Survey Analysis – Python & Power BI
This project explores and visualizes insights from a global Data Professionals Survey. Using Python, raw survey data was cleaned and processed before being visualized in a rich Power BI dashboard. The goal was to uncover key trends in the data profession landscape, including salary, tools used, job satisfaction, and regional demographics.

📁 Project Structure
Data_professionals_raw_data.csv – Original dataset
Exploring_data_using_python - python scripts for cleaning data
Cleaned_data.csv – Cleaned version
Data professionals analysis.pbix – Final Power BI dashboard
README.md – Project summary and documentation

🧰 Tools & Technologies
Python (pandas) – Data exploration, cleaning, and preprocessing
Power BI Desktop – Dashboard creation and interactive data analysis
