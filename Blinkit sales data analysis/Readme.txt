🛒 Blinkit Sales Analysis – Power BI Dashboard
This project is a Power BI sales analysis of data from Blinkit, focusing on performance metrics across product categories, fat content, outlet types, and regions. The dashboard provides insights into total sales, average sales, ratings, and product visibility, helping stakeholders make data-driven decisions.

📊 Dashboard Overview
Key metrics visualized in the Power BI report:

Total Sales: $1.20M

Average Sales: $140.99

Number of Items Sold: 8.52K

Average Rating: 3.92

🔍 Key Insights
Top Performing Outlet Type:

Supermarket Type1 leads with over $787K in sales and more than 5,500 items sold.

Sales by Fat Content:

Low Fat items outperformed Regular with $776K in total sales.

Item Category Trends:

Leading item types include Fruits, Snacks, Household, and Frozen Foods.

Outlet Establishment Timeline:

Sales growth correlates with outlets established post-2010, peaking around 2015-2020.

Sales by Location Tier:

Tier 3 cities recorded the highest sales volume, followed by Tier 2 and Tier 1.

🧩 Filters & Slicers
The dashboard includes dynamic filters to segment data by:

Outlet Type

Outlet Size

Outlet Location

These allow users to interactively explore patterns and tailor the analysis to specific business questions.
